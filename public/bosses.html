Introduction

Welcome to the official documentation for Bloodborne's API! Whether you're a developer, gamer, or enthusiast, this documentation provides comprehensive information on accessing and utilizing Bloodborne's API to enhance your experience.

All interactions with the API are performed via GET requests, ensuring simplicity and ease of use.

REST

Base url: https://bloodborne-simple-api-dev-sdqb.2.us-1.fl0.io/api

The base URL for Bloodborne's API serves as the entry point for accessing all available resources.Expect all responses from the API to be formatted in JSON allowing for easy integration into your applications.

  
    
GET https://bloodborne-simple-api-dev-sdqb.2.us-1.fl0.io/api

  

  

    {
      "Bosses" : "https://bloodborne-simple-api-dev-sdqb.2.us-1.fl0.io/api/bosses" ,
      "Charactes" : "https://bloodborne-simple-api-dev-sdqb.2.us-1.fl0.io/api/characters" 
    }

Bloodborne Bosses

There are 4 bosses in total, ordered by ID.

Bosses schema

Key Type Description
id int
name string
information string
location object
drops object
image string(url)
fight string

Get all bosses

You can access the bosses list via the /bosses endpoint.

          
            
GET https://bloodborne-simple-api-dev-sdqb.2.us-1.fl0.io/api/bosses

Get a single boss

You can get an individual character by appending the ID as a parameter: /bosses/2

            
              
GET https://bloodborne-simple-api-dev-sdqb.2.us-1.fl0.io/api/bosses/2
